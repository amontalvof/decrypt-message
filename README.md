# Decrypt App

## Frontend Technologies

-   TypeScript
-   React
-   React Query
-   Styled Components

## Backend Technologies

-   TypeScript
-   Node.js
-   Express.js

## Database

-   MySQL

## Testing

- Jest
- Cypress

> Note: You need to add a .env file with the followings environments on the backend side to be able to connect to de DB

```
DB_HOST=localhost
DB_USER=root
DB_PASS=password
DB_NAME=decrypt_db
DB_PORT=3306
PORT=8080
REACT_APP_URL=http://localhost:3000
```
